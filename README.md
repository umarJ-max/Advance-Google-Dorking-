# Google Dorking Advanced Tool

An intelligent search tool that enhances queries with Google dorking operators for advanced search capabilities.

## Features

- Intelligent query analysis and enhancement
- Automatic Google dork generation
- Quick filter buttons for common searches
- Copy-to-clipboard functionality
- Responsive design

## Deployment

### Vercel Deployment

1. Install Vercel CLI:
   ```bash
   npm i -g vercel
   ```

2. Deploy to Vercel:
   ```bash
   vercel
   ```

3. Follow the prompts to configure your deployment.

## Local Development

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Run the application:
   ```bash
   python app.py
   ```

3. Open http://localhost:5000 in your browser.

## Usage

1. Enter your search query in the main search box
2. Optionally specify a website to search within
3. Click "Search" or use one of the quick filter buttons
4. View the enhanced query and click "Open in Google" to search

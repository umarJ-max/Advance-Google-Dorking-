Find sensitive documents, backups, and exposed file directories.
intitle:"index of"
Open Directory Listing
Finds web servers with directory browsing enabled. You can explore the full file structure of the server and often find sensitive documents.
intitle:"index of" "parent directory"
Parent Directory Access
Targets open directories showing the "parent directory" link — strong indicator of unrestricted access to the server file system.
filetype:pdf "confidential" OR "internal use only"
Confidential PDF Documents
Searches for PDF files labeled confidential or internal. Companies often accidentally expose internal reports, memos, and data to Google indexing.
filetype:xls OR filetype:xlsx intext:password
Excel Files with Passwords
Finds spreadsheet files that contain the word "password" in their content — often credential sheets, IT inventory files, or key management docs.
filetype:sql "INSERT INTO" "users"
SQL Database Dumps
SQL dump files exported from databases. These files often contain usernames, hashed or plaintext passwords, and full database schemas.
filetype:bak OR filetype:backup OR filetype:old
Backup Files
Backup copies of files often contain older config data, credentials, or source code. Developers frequently forget to restrict access to these.
filetype:log inurl:"password.log"
Password Log Files
Log files that explicitly contain cleartext usernames and passwords. These are sometimes generated by misconfigured applications.
filetype:dat "password.dat"
Password Data Files
Finds .dat files named password.dat — these often contain plaintext username/password combinations used by older applications.
filetype:wab wab
Outlook Address Books
Microsoft Outlook WAB (Windows Address Book) files. Contains contact details, email addresses, phone numbers — useful for OSINT on individuals and organizations.
"# Dumping data for table" "users"
MySQL User Table Dumps
Exported MySQL dumps with user tables. These contain full database records including usernames, emails, and hashed or plaintext passwords.
filetype:mbx intext:Subject
Mailbox Files (MBX)
Outlook v1-4 or Eudora mailbox files exposed on the web. These contain email content, sender addresses, and other communication metadata.
e-mail address filetype:csv
Exported Email Lists (CSV)
Email contact lists exported in CSV format. Often dumped from CRM systems or mailing software — can contain names, phone numbers, and full contact details.
Locate exposed admin panels, login portals, and authentication pages.
inurl:admin/login OR inurl:admin/signin
Admin Login Pages
Finds admin panel login pages. Many use default credentials or are vulnerable to brute force. Combine with site: to target a specific domain.
intitle:"Login" inurl:login.php
PHP Login Pages
Targets PHP login forms. PHP pages are often unpatched and vulnerable to SQL injection or weak password authentication.
intitle:"phpMyAdmin" "Welcome to phpMyAdmin"
phpMyAdmin Panels
Database admin panels with public access. Many are left open with default credentials (root with no password). Direct database access = full data exposure.
inurl:"exchange/logon.asp" OR intitle:"Outlook Web Access"
Outlook Web Access (OWA)
Microsoft Exchange email login portals. OWA exposes the exchange server and can be targeted for password spraying attacks to gain internal email access.
inurl:root.asp?acs=anon
OWA Public Folders (Anonymous)
Outlook Web Access public folders accessible anonymously. Allows browsing the Exchange address book and public folders without any login credential.
intitle:"ZyXEL Prestige Router" "Enter password"
ZyXEL Router Login
Authentication screen for ZyXEL Prestige routers exposed to the internet. Default credentials are commonly unchanged — admin/1234 or admin/admin.
intitle:"Dell Remote Access Controller"
Dell Remote Access Controller
DRAC allows full out-of-band remote administration of Dell servers. If exposed, attackers gain OS-level control even if the server is offline.
"please log in" inurl:login
Generic Login Portals
Casts a wide net for login forms using common phrasing. Good starting point for reconnaissance when you don't know what platform the target uses.
filetype:reg reg +intext:"defaultusername" +intext:"defaultpassword"
Windows Registry Credential Files
Exported Windows Registry files containing saved default usernames and passwords. These are often left exposed on public shares or FTP servers.
inurl:metaframexp/default/login.asp
Citrix Metaframe Login
Citrix remote access portals. Insecure configurations allow attackers to profile a network and potentially access internal applications remotely.
Discover exposed databases, connection files, and database management interfaces.
filetype:inc intext:mysql_connect
PHP Database Connection Files
.INC files contain PHP code with unencrypted database credentials. The mysql_connect function handles authentication — exposing host, username, and password.
intitle:"index of" mysql.conf OR mysql_config
MySQL Configuration Files
MySQL config files expose port numbers, version info, and server paths. Combined with other dorks, attackers can map the full database setup.
filetype:sql +password
SQL Files with Passwords
Database SQL files containing the word "password". These are often migration scripts or backups with embedded credentials or user tables.
filetype:sql +IDENTIFIED BY -cvs
SQL GRANT Statements
SQL scripts using IDENTIFIED BY to set database user permissions. These often contain plaintext or encrypted passwords used for database authentication.
inurl:"/database/comersus.mdb"
Comersus Shopping Cart Database
Access database file from the Comersus e-commerce platform. Contains full customer records, order history, and payment data from online stores.
filetype:mdb inurl:users.mdb
Access User Databases
Microsoft Access databases named "users.mdb" used as password stores. Many are completely unprotected and downloadable directly.
filetype:ldb admin
Access Lock Files (.ldb)
Lock files always share the same name as their database. By replacing .ldb with .mdb in the URL, attackers can directly download the parent database.
intitle:"index of /phpmyadmin"
phpMyAdmin Index
Open directory listing for phpMyAdmin installations. From here, attackers can access the full database management UI and run arbitrary SQL queries.
Find exposed configuration files that reveal server setup, credentials, and architecture.
inurl:php.ini filetype:ini
PHP Configuration (php.ini)
PHP's main configuration file. Contains default database credentials, server paths, error settings, and sometimes sensitive initialization variables.
filetype:conf inurl:firewall -intitle:cvs
Firewall Configuration Files
Firewall config files reveal allowed/denied IP rules, ports, and network topology. Attackers use this to identify open paths through the firewall.
filetype:conf inurl:proftpd.conf -sample
ProFTPD FTP Configuration
FTP server configuration exposing installation paths, log file locations, generic usernames, and associated group names for the FTP service.
filetype:conf slapd.conf
OpenLDAP Configuration
Configuration for OpenLDAP server including the root password in cleartext, along with full paths and permissions for all related applications.
inurl:"slapd.conf" intext:"rootpw"
LDAP Root Password
Specifically targets slapd.conf files that contain the rootpw field — the root password for the entire LDAP directory, in cleartext or hashed form.
filetype:ini ws_ftp pwd
WS_FTP Credentials
WS_FTP config files using an extremely weak encryption method for stored passwords. These are easily crackable using publicly available tools.
filetype:pem intext:private
Exposed Private Keys (.pem)
Private key files used for SSL/TLS encryption. If exposed, attackers can decrypt all traffic to/from the server and impersonate it entirely.
filetype:cfg ks intext:rootpw -sample
Linux Kickstart Root Passwords
Anaconda (Linux installer) config files sometimes contain root passwords in cleartext or MD5 hash. These are used for automated server provisioning.
inurl:zebra.conf intext:password -sample
GNU Zebra Routing Config
GNU Zebra manages TCP/IP routing protocols. Config files contain enable passwords and full routing topology — same format as Cisco IOS configs.
+htpasswd +WS_FTP.LOG filetype:log
FTP Logs with Auth Paths
WS_FTP log files revealing server paths, combined with htpasswd references. Often leads to discovering the location of authentication credential files.

Pages showing SQL error messages are strong indicators of SQL injection vulnerabilities. These errors expose database structure, table names, and query parameters.

ColdFusion SQL Errors
ColdFusion error pages logging SQL SELECT or INSERT statements. These reveal full file paths and exact injection parameters on the server.
intext:"Warning: Failed opening" "include_path"
PHP File Include Errors
PHP warning messages revealing application paths, filename structures, and include paths — useful for identifying local file inclusion vulnerabilities.
"Request Details" "Control Tree" "Server Variables"
ASP.NET Trace Pages
ASP.NET trace pages expose session IDs, stack traces, port numbers, IP addresses, full paths, and HTTP headers — a goldmine for attack planning.
filetype:php inurl:"viewfile" -"index.php"
PHP File Viewer Scripts
PHP scripts designed to view server files via URL parameters. Often vulnerable to directory traversal attacks — try appending /../../../etc/passwd to the URL.
intitle:"Index of /" modified php.exe
PHP-CGI on Windows Apache
PHP installed as CGI on Windows Apache allows arbitrary file reads via path injection: /php/php.exe?c:\boot.ini — a classic arbitrary file disclosure.
intitle:"htsearch error" ht://Dig error
ht://Dig Search Errors
ht://Dig errors expose admin email, cgi-bin path, directory structure, and database locations. Older versions (3.1.1–3.2) vulnerable to directory traversal.
intitle:"index of" inurl:recycler
Exposed Windows Recycle Bin
Default Windows recycle bin folder exposed on a web server. Files here may contain deleted sensitive documents, and SIDs (Security Identifiers) of users are revealed.
filetype:log access.log -CVS
HTTP Access Logs
Web server access logs containing full request history — usernames, passwords from basic auth, trusted machine IPs, full server paths, and query parameters.
"Invision Power Board Database Error"
Invision Board SQL Errors
SQL error messages from Invision Power Board forums including connection errors, access denied messages, and full PHP file paths on the webserver.
